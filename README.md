# Pousada Bela Silvia — site estático

Estrutura pronta para GitHub Pages:

- `index.html`
- `assets/images/`

## Publicação

1. Envie `index.html` para a raiz do repositório.
2. Envie a pasta `assets` completa, mantendo exatamente os nomes e subpastas.
3. No GitHub: **Settings → Pages → Deploy from a branch → main / root → Save**.
4. Aguarde a publicação e abra o endereço exibido pelo GitHub Pages.

Não renomeie arquivos individualmente: o HTML usa caminhos relativos e diferencia letras maiúsculas de minúsculas.
